#### **MoveSure** — AI-Based Posture Monitoring System
*Developed a real-time, AI-driven posture tracking system using computer vision and deep learning to monitor and correct ergonomics.*

* **Core Contributions:** Co-authored the system architecture and high/low-level design blocks; engineered the core logic workflows for real-time camera-based posture detection; and designed the data management pipelines.
